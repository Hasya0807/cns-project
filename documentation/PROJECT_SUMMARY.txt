🔐 ENCRYPTION FEATURES

• AES-256-CBC Client-Side Encryption
  - Messages encrypted before leaving browser
  - Key stored in sessionStorage
  - Automatically cleared when tab closes

• AES-256-GCM Server-Side Encryption  
  - Additional security layer
  - Authentication tag for integrity
  - Hash verification

• Result: ZERO KNOWLEDGE ENCRYPTION
  - Server cannot read messages
  - Database cannot read messages  
  - Only sender & recipient can decrypt
  - Perfect end-to-end privacy
